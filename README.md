# pdf-merger-python-ex-8
# PDF Merger Project

## Overview

This project is a simple PDF Merger application built using Python. The application allows users to combine multiple PDF files into a single PDF document.

The project is beginner-friendly and helps in understanding:

* File handling in Python
* Working with external libraries
* PDF processing
* Basic project structure

---

# Features

* Merge multiple PDF files into one
* Simple and easy-to-use code
* Lightweight project
* Beginner-friendly implementation

---

# Technologies Used

* Python 3
* PyPDF2 Library

---


# Installation

## Step 1: Clone the Repository

```bash
git clone https://github.com/your-username/pdf-merger.git
```

## Step 2: Move into the Project Folder

```bash
cd pdf-merger
```

## Step 3: Install Dependencies

```bash
pip install PyPDF2
```

---

# How to Run the Project

Run the following command:

```bash
python main.py
```

---



# Output

After running the program:

* All selected PDF files will be merged
